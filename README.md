# 🌐 Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, and contact details in a clean and responsive design.

## 🚀 Features

* Responsive design (works on mobile & desktop)
* Smooth scrolling and navigation
* Projects showcase section
* About me section
* Contact form / contact details
* Simple and clean UI
* Toggle button
* 
## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript 

## 📂 Project Structure

```
portfolio/
│── index.html
│── style.css
│── script.js

```

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/yash3556/yash-portfolio-website
```
2. Open the folder

3. Run the project:

* Open `index.html` in your browser
